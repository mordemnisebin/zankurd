# ZanKurd

ZanKurd; Kürtler, Kürdistan coğrafyası, sanat, kültür, yemek, dil, tarih ve demokratik özgürlük paradigması gibi başlıklarda çoktan seçmeli sorularla öğrenmeyi destekleyen bir eğitim uygulaması için hazırlanmış açık kaynak proje deposudur.

Bu ilk commit, depoyu başlatmak için oluşturuldu. Uygulama iskeleti ve geliştirme belgeleri ayrı bir geliştirme dalında hazırlanacaktır.
