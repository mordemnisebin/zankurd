## Task 4: Quiz Result Screen Integration

**Objective:** When user finishes a contest quiz, reward is claimed and banner shown.

### Files to Modify
- `lib/src/screens/quiz_result_screen.dart` — Add conditional contest reward banner

### Implementation
1. **Result Screen Props:**
   - Add optional `contestId` parameter (null = regular quiz, non-null = contest quiz)
   
2. **Reward Claim Flow:**
   - After result computed, if `contestId` != null:
     - Call `repository.claimContestReward(contestId)`
     - Extract rank reward from response
     - Show banner with icon + text + badge teaser

3. **Conditional Banner:**
   ```dart
   if (widget.contestId != null) {
     final reward = await repository.claimContestReward(widget.contestId!);
     final rankReward = reward?['rank_reward'] ?? 0;
     final badgeSlug = reward?['badge_awarded'];
     // Show banner: "🏆 +{rankReward} Altın! 🥇 {badge_name}"
   }
   ```
